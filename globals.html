bootstrapIpcApi

  • bootstrapIpcApi(dependencies: AppDependencies, options?: AppOptions): Container
    • Defined in src/lib/bootstrap.ts:63

    Bootstrap the app and return an augmented inversifyjs container

    example
    import { bootstrapIpcApi } from 'electron-api-ipc';

const app = bootstrapIpcApi({ controllers: [...] });

app.listen();

// ready to receive events from renderer process

    Parameters

    • dependencies: AppDependencies

      the dependencies of the app: controllers and services, services must be inversifyjs ready

    • Optional options: AppOptions

      a small set of options to customize the behavior (logger, mock ...)

    Returns Container
